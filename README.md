# Nova Security Platform 

**Nova** is an intelligent security architect designed to bridge the gap between small business infrastructure and advanced threat detection.

##  Core Module: Nova Intelligence
The current iteration uses an **Isolation Forest** algorithm to detect network anomalies. 

### Features:
- **Behavioral Analysis:** Moves beyond static signatures to identify "unusual" traffic patterns.
- **Unsupervised Learning:** Automatically clusters normal traffic vs. potential threats.
- **Visual Feedback:** Generates scatter plots for architectural review.
